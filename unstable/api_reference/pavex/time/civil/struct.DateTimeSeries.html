Struct DateTimeSeries 

pub struct DateTimeSeries { /* private fields */ }
Expand description

An iterator over periodic datetimes, created by DateTime::series.

It is exhausted when the next value would exceed the limits of a Span or DateTime value.

This iterator is created by DateTime::series.
